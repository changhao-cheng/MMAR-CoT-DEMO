# MMAR CoT Demo

This repository hosts a public demo page for a curated subset of 30 MMAR audio samples.

## Demo
GitHub Pages: `https://changhao-cheng.github.io/MMAR-CoT-DEMO/`

## About
This project visualizes selected MMAR samples, including:
- question
- options
- answer
- thinking process
- cue
- rubrics
- audio playback

## Source
- Paper: [MMAR: A Challenging Benchmark for Deep Reasoning in Speech, Audio, Music, and Their Mix](https://arxiv.org/abs/2505.13032)
- GitHub: `https://github.com/ddlBoJack/MMAR`

## Note
This repository only contains a small curated demo subset for visualization purposes and does not represent the full MMAR benchmark.
